# FireWatch User Guides

This section contains guides for Security Operations Center (SOC) analysts and operators
who are learning or using FireWatch. All guides are written for a junior analyst — you
know security basics, and you are new to this tool.

---

## Start here

- [Getting Started](getting-started.md) — Your first 10 minutes with FireWatch: a guided
  path from the Dashboard through to connecting a source.

---

## Page guides

One guide per page in the FireWatch interface, each following the same five-section
structure: what the page is for, what you are looking at, how to read it, what you would
do here, and terms used.

- [Dashboard](dashboard.md) — Shift-start screen; triage queue, threat actors, activity
  timeline, and recommended actions.
- [Network Logs](network-logs.md) — Raw event feed; filter bar, entity graph, and per-row
  detail for deep investigation.
- [AI Engine](ai-engine.md) — Audit surface for local AI verdicts; confidence, score
  effects, feedback, and model drift.
- [Threat Intelligence](threat-intelligence.md) — Geographic and network-level view of
  attack sources; country map and ASN rankings.
- [Settings](settings.md) — Connect data sources, configure the local AI model, tune
  escalation and notification thresholds.
